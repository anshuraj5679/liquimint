# Security Package

This folder contains the pre-judge security package for LiquiMint:

- `THREAT_MODEL.md`: trust boundaries, threat scenarios, and mitigations.
- `MANUAL_AUDIT_NOTES.md`: prioritized findings and remediations.
- `SLITHER_REPORT.md`: static analysis workflow and result capture.
- `POST_DEPLOY_VERIFICATION.md`: reproducible on-chain verification steps.

Scope of this package is the current repository revision as of 2026-03-01.
